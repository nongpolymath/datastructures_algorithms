# Data Structures & Algorithms (Python)

A personal learning repository covering core data structures and algorithms implemented in Python, including practice files and interview question solutions.

## Project Structure

```
src/
├── arrays/                  # Array manipulation and 2D arrays
├── binary_heap/             # Min/max heap implementation
├── binary_search_tree/      # BST with practice exercises
├── dictionaries/            # Dictionary-based problems
├── graph/                   # Graph with BFS and DFS traversals
├── linked_lists/            # Singly, doubly, and circular linked lists
├── queues_lib/              # Queue implementations (list, linked list, circular)
├── recursion/               # Recursion problems and practice sets
├── searching/               # Binary search
├── sorting/                 # Sorting algorithms
├── stacks/                  # Stack implementations and interview problems
├── trees/                   # Binary trees (list and linked list), AVL tree
├── tries/                   # Trie implementation
├── InterviewQuestions.py
├── interviewQuestionsBigO.py
└── timeComplexities.py
```

## Topics Covered

**Data Structures**
- Arrays, Lists
- Linked Lists (singly, doubly, circular)
- Stacks & Queues
- Binary Search Tree
- Binary Heap
- AVL Tree
- Binary Tree
- Trie
- Graph

**Algorithms**
- Sorting: Bubble, Selection, Insertion, Merge, Quick, Heap
- Searching: Binary Search
- Graph traversal: BFS, DFS
- Recursion

## Setup

Requires Python 3.8+.

```bash
# Install as an editable package (optional)
pip install -e .
```

## Usage

Each file is standalone and can be run directly:

```bash
python src/binary_search_tree/binary_search_tree_bst.py
```
