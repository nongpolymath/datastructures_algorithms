# Important Note

# In Python, “push” (or enqueue) always goes onto the “back” (aka tail/rear) of the structure, 
# but what counts as the “back” depends on whether you’re implementing a stack (LIFO) or a queue (FIFO)

# For Stacks:
# Push (push or append) → end of the list.
# Pop → end of the list.

# For Queue:
# Enqueue (append) → right end.
# Dequeue (popleft) → left end.
